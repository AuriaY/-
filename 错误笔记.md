## 错误笔记

1. 矩阵/图

   ```c++
   /****
   编程中的矩阵和数学里的坐标系是不同的
   如果说要像什么的话，应该 是行列式
   先是行标i,再到列标j
   所以想要矩阵的边缘的时候，要注意边界限制n,m是行的还是列的
   ****/
   for(int i = 0; i < n; i++)
   {	//遍历行，判断左右边缘两列
       if(g[i][0])
       if(g[i][m - 1])
   }
   for(int i = 0; i < m; i++)
   {	//遍历列，固定行，判断上下边缘两列
       if(g[0][i])
       if(g[n - 1][i])
   }
   ```
